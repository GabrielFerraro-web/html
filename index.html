Comandos de programação

Diz sobre os principais comandos do HTML e do CSS, exemplo: "h1" "p" "img"

Comando "h1"

É usado para escrever títulos, sendo sempre utilizado para o título principal

Comando "P"

Define os parágrafos. Uma das das tags mais básicas e essenciais para conversão de texto.

Comando "img"

utilizada para colocar uma imagem no texto. Requer o comando: "img src="nome-da-imagem.jpg"

Formatação de cor

É utilizada para definir a cor de texto com a propiedade "color" da seguinte maneira: "color: blue;" as cores devem ser escritas em inglês ou pelo "rgb(255, 0, 0);" que é #000000

Formatação de centralização

É utilizada a propiedade "text-align: center;" para definir sua centralização

Formatação de tamanho

Usada para estlizar o texto com a propiedade "font-size: númeropx;"
